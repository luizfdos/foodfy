## :construction_worker: Como usar

- Necessário NodeJS

- Clonar este respositório:
  ```
  $ git clone https://github.com/luizfdos/foodfy.git
  ```
- Instale as dependências:
  ```
  $ npm install
  ```
- Inicie o servidor:
  ```
  $ npm start 
  ```
- Acesse pelo navegador o caminho:
  ```
  http://127.0.0.1:5000/
  ```
